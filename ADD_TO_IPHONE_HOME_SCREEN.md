# How to Add Widget to iPhone Home Screen

## 📱 Step-by-Step Instructions

### Step 1: Open on Your iPhone

1. **Make sure your iPhone is on the same Wi-Fi network as your Mac**

2. **Open Safari** on your iPhone (not Chrome or other browsers)

3. **Type in the address bar**:
   ```
   http://192.168.68.81:5002
   ```

4. **Tap Go** - You should see the same widget interface

### Step 2: Add to Home Screen

1. **Tap the Share button** at the bottom of Safari
   - It looks like a square with an arrow pointing up
   - On newer iPhones, it's at the bottom center
   - On older iPhones, it's at the bottom right

2. **Scroll down** in the share menu

3. **Tap "Add to Home Screen"**
   - It has a plus icon (+)
   - It's in the bottom row of the share menu

4. **Customize the name** (optional)
   - Default: "PDF Monitor"
   - You can change it to whatever you want

5. **Tap "Add"** in the top right

### Step 3: Use Your Widget!

- The widget icon will appear on your iPhone home screen
- Tap it anytime to control your PDF Monitor
- It works just like a native iOS app!

## ⚠️ Important Notes

- **Must use Safari** - Chrome doesn't support "Add to Home Screen"
- **Must be on same Wi-Fi** - iPhone and Mac need same network
- **Use the exact IP**: `192.168.68.81:5002`
- **Include http://** - Don't forget the protocol

## 🐛 Troubleshooting

**Don't see "Add to Home Screen" option?**
- Make sure you're using Safari (not Chrome)
- Make sure you're on the actual website (not a bookmark)
- Try refreshing the page first

**Can't connect from iPhone?**
- Check iPhone and Mac are on same Wi-Fi
- Try the IP address again: `192.168.68.81:5002`
- Make sure server is still running on Mac

**Widget doesn't work after adding?**
- Tap the widget icon on home screen
- It should open Safari and load the interface
- If it doesn't, try removing and re-adding it
